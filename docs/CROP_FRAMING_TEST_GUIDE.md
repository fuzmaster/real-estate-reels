# Crop + Framing Test Guide

## Goal

Stop wide MLS photos from getting destroyed in a vertical 9:16 reel.

## Use these rules

### Use "Fill Screen" for:
- exterior front photos
- drone photos
- aerial photos
- backyard photos
- detail shots where cropping is okay

### Use "Show Whole Room" for:
- living rooms
- kitchens
- bedrooms
- dining rooms
- any horizontal photo where buyers need to understand the full room

## Test checklist

Before rendering:

1. Put the best exterior/drone/living room image first.
2. Click **MAKE HERO** on that image.
3. Set the hero image to **Fill Screen**.
4. Set wide interiors to **Show Whole Room**.
5. Set close-up/detail shots to **Fill Screen**.
6. For Fill Screen photos, test Left / Center / Right and Top / Middle / Bottom.
7. Keep duration at 15 seconds.
8. Use Social Punchy + Fast Cuts for the first test.
9. Render.
10. Send the MP4 to Gemini with the included review prompt.

## What good looks like

The viewer should understand:
- what the house looks like
- what the main rooms look like
- who the agent is
- what action to take next

If the viewer only sees chopped pieces of furniture, the crop mode is wrong.
