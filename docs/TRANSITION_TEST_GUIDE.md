# V6 Transition Test Guide

## Test 1 — Clean social version

```txt
Duration: 15 seconds
Pacing: Fast
Style: Social Punchy
Transition: Smart Mix
Auto Enhance: ON
Persistent Branding: ON
Progress Bar: ON
```

Expected result:
- No black opener
- Photos feel more alive
- Text pops in with better motion
- Room photos still need correct framing, but the edit should feel less dead

## Test 2 — Brokerage-safe version

```txt
Style: Brokerage Clean
Pacing: Balanced
Transition: Soft Fade
Progress Bar: OFF
```

Expected result:
- Cleaner and less aggressive
- Better for older agents or conservative brokerages

## Test 3 — High-energy social version

```txt
Style: Social Punchy
Pacing: Fast
Transition: Whip Pan
Progress Bar: ON
```

Expected result:
- More TikTok/Reels energy
- May feel too intense for luxury listings

## What to watch for

Bad:
- Kitchen still shows only appliance
- Living room still shows only couch
- Photos still look dead
- Transitions distract from house
- Text blocks important property detail

Good:
- Viewer understands the house faster
- Motion feels intentional
- Hook is readable in less than one second
- First photo makes sense
- Final CTA feels like a real ad, not a slideshow end card
